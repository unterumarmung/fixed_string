Returns the number of `TChar` elements in the fixed string, i.e. `std::distance(begin(), end())`.

:::info note
For `fixstr::fixed_string`, the elements are bytes (objects of type `char`), which are not the same as characters if a multibyte encoding such as UTF-8 is used.
:::

## Parameters
(none)

## Return value
The number of `TChar` elements in the fixed string.

## Complexity
Constant.
