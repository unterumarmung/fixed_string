# `fixstr::basic_fixed_string::max_size`

Returns the maximum number of elements the fixed string is able to hold due to system or library implementation limitations, i.e. `std::distance(begin(), end())` for the largest fixed string.

:::info note
Because each `fixstr::basic_fixed_string<TChar, N, TTraits>` is a fixed-size container, the value returned by `max_size` equals `N` (which is also the value returned by [`size`](./size));
:::

## Parameters
(none)

## Return value
Maximum number of characters.

## Complexity
Constant.
